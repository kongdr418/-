# ProcessKiller 项目重要提醒

## 法律与安全警示

本项目仅供学习，请在完全合法场景下使用


## 核心功能说明
```
├── 进程监控核心
│   ├── 实时进程扫描（每秒1次）
│   ├── 强制终止进程（NT特权级）
│   └── 开机自启动注册
├── EXE生成器
│   ├── 自定义进程名匹配
│   ├── 持久化模块打包
│   └── 数字签名支持
```

## 使用步骤
  环境准备
  需要安装Pyinstaller
```
pip install pyinstaller --upgrade
```
